O que aprendemos até aqui em IDW⛅

Na diciplina de Introdução ao Desenvolvimento Web estudamos as tecnologias Web básicas relacionadas ao desenvolvimento de interfaces de sites e sistemas Web. Até aqui estudamos os seguintes conceitos HTML:

  • O que é HTML?
  • Estrutura básica de um documento HTML
    • tag <head>
    • tag <body>
    • declaração <!DOCTYPE html>
    • atributo lang na tag <html>
  • Tags de título: <h1> a <h6>
  • Tags de ênfase e destaque: <strong> <em> <ins> <del> <sup> <sub> <small> <mark>
  • Tags de formatação textual simples: <b> <i> <u> <s>
